# Interactive MCS Simulator

This directory will contain a browser-based interactive simulator for the Monte-Carlo ROI model.

## Planned Features
- Adjust input parameters (CAPEX, OPEX, Volume, Oil Price) via sliders
- Run simulation with configurable iterations
- Display real-time ROI histogram
- Show key statistics (Mean, VaR, P(Loss), etc.)
- Tornado diagram for sensitivity analysis

## Technology Stack
- HTML5 + JavaScript (no server required)
- Plotly.js for interactive charts
- Pure JavaScript for sampling (no Python dependency)
